Introducción a HTML

    Mediante HTML se define la estructura de las páginas web y se asignan formatos a los contenidos para hacerlos más legibles. Permite mostrar el contenido de periódicos, recorrer museos, gestionar tiendas, etc.

    Para producir documentos HTML sirve un simple editor de textos. Obviamente, existen editores de html que facilitan y aceleran el manejo de estos documentos.

    La primera línea de un fichero codificado para HTML 5 debe ser:

       <!DOCTYPE html>

    Para saber si un navegador soporta alguna de las características de HTML puede utilizarse la página:

       http://www.html5test.com

    HTML 5 incorpora respecto a HTML 4 los elementos:
  •     article.
  •     aside.
  •     hgroup.
  •     header.
  •     footer.
  •     nav.
  •     figure.
  •     section.
  •     video.
  •     audio.
  •     embed.
  •     mark.
  •     progress.
  •     meter.
  •     time.
  •     canvas.
  •     command.
  •     details.
  •     output.
  •     input.
    Además, ha cambiado la funcionalidad de algunos elementos:
  •     a.- sin el atributo href actuará como una reserva de espacio para colocar un enlace.
  •     address.- ahora participa del nuevo concepto de sección.
  •     cite.- representa simplemente el título de una obra.
  •     menu.- se redefine para ser utilizado en barras de herramientas y menús de contexto.
  •     s.- representa contenido que ya no es fiable o importante.
  •     small.- representa una impresión pequeña para comentarios colaterales o letra pequeña de temas legales.
  •     head.- no permite tener como hijo un elemento object.
    Y desaparecen:
  •     Por ser mejor resueltos mediante css: basefont, big, center, font, strike, tt y u.
  •     Por perjudicar la usabilidad y la accesibilidad.- frame, frameset y noframes.
  •     Por resultar confusos o reemplazados por otros.- acronym, applet, isindex y dir.
