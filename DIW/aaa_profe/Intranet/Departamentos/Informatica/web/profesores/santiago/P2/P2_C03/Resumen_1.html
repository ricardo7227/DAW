Valores de las propiedades

Los valores que se pueden asignar a las diferentes propiedades pueden ser de diferente tipo:

  • Palabras clave, por ejemplo: top, bottom, red, etc.
  • Texto.-
  • Número.-
  • Número con unidad de medida, por ejemplo: 3cm.
  • Medida de pantalla en pixels.
  • Medida con relación a un tamaño de fuente: em y ex.
  • Ángulos (grados y radianes).
  • Valores de tiempo (segundos y milisegundos).
  • Frecuencias (Hz y KHz).
  • Valores de porcentaje, por ejemplo: 40%.
  • Función de color, por ejemplo: rgb(0,255,0).
  • Función de url, por ejemplo: url(www.xxx.com).

Unidades de medida

Las unidades se especifican con un signo - o + (opcional), un número y el nombre de la unidad, sin espacios por medio. Por ejemplo:


          height: -30%;

Las unidades de medida absoluta que se pueden utilizar son:

  • in.- pulgadas. 1 pulgada equivale a 2,54 cm.
  • cm.- centímetros.
  • mm.- milímetros.
  • pt.- puntos. 1 pulgada equivale a 72 puntos.
  • pc.- picas. 1 pica equivale a 12 puntos.

Como unidades de medida relativas están:

  • ex.- tamaño equivalente a la altura de la letra x (minúscula) de la fuente asociada al elemento.
  • em.- tamaño equivalente a la altura de la letra M (mayúscula) de la fuente asociada al elemento.
  • px.- pixel, relativo a la resolución del área de visualización.
  • %.- porcentaje con relación al elemento padre. Ver ejemplo en pag. 233 (Pag. 35 de la parte 2).

Valores de color

Un valor de color puede ser una palabra clave (red, ...) o una especificación numérica RGB. Las tres formas de especificar el color RGB son:

  • #rrvvaa.- con valores de 00 a FF.
  • rgb(r,v,a).- con valores de 0 a 255.
  • rgb(r%,v%,a%).- con valores de 0 a 100.

Los colores RGBA son una extensión de los RGB que incorporan un cuarto parámetro, que indica el canal alfa o grado de opacidad del objeto, comprendido entre 0.0 (transparencia máxima) y 1.0 (opacidad máxima). Un ejemplo sería:


          .texto  {color: rgb(255,255,255,0.3);}

La medida HSL (hue, saturation y lightness) especifica el color mediante valores de tono, saturación y luminosidad, através de la función hsl(). El tono varía entre 0 y 360, formando una rueda de colores, siendo el 0 el rojo, el 120 el verde y el 240 el azul. La saturación es un porcentaje donde 0% significa una sombra de grises y 100% es color completo. La luminosidad es otro porcentaje donde 0% es máxima oscuridad y 100% máxima luminosidad. Un ejemplo sería:


          .texto  {color: hsl(128,80%,70%);}

La medida HSLA incorporaría un cuarto parámetro (entre 0.0 y 1.0) con el valor de la opacidad.

Valores de url

Para expresar una dirección URL en CSS se puede utilizar la forma url(Dirección_url), donde la dirección URL puede encerrarse entre comillas dobles o simples.

Cuando dentro de la dirección URL aparecen paréntesis, espacios, comas o comillas deben codificarse con el carácter de escape (\).

Una dirección relativa se interpreta como relativa respecto al archivo de hoja de estilo, no al documento html.

Algunos ejemplos:

          body {background: url(imagen.jpg);}
          body {background: url(http://www.tachan.es/imagen.jpg);}
          body {background: url("imagen.jpg");}

Valores de texto y palabras clave

Cuando el valor lleva espacios debe encerrarse entre comillas. Algún ejemplo:

          p {font-family: "Times New Roman", Times, sherif;}
          p:before {content: "Nota adiccional: ";}
