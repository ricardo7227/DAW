CSS y HTML dinámico

    Desde código script se puede actuar sobre los estilos CSS a través del modelo DOM que permite acceder a las partes de un documento desde un programa, es decir, desde fuera del código HTML.

    A las propiedades CSS se accede generalmente de la misma manera, utilizando su nombre de propiedad, por ejemplo, color. Cuando el nombre de la propiedad contiene algún guión, para usar esa propiedad en un script se le debe eliminar ese guión y cambiar a mayúsculas la letra que sigue al guión, por lo tanto, font-family dentro de un guión se convierte en fontFamily. Así, para acceder a la propiedad font-family de un elemento con identificador id1  puede utilizarse el código:

          id1.style.fontFamily = "Arial";

    Esta instrucción JavaScript debe formar parte de alguna función del script de la página, por ejemplo:

          . . .
          <head>
          . . .
             <script>
                function CambiarFuente(obj) {
                                obj.style.fontSize = "xx-large";
                }
             </script>
          </head>
          . . .
             <div  onClick="CambiarFuente(this);">
          . . .

    Así, al hacer click sobre el texto se producirá un cambio de tamaño de la fuente.

    Se puede observar que se utiliza el evento onClick() para llamar a una función de JavaScript que está definida dentro de un bloque <script>...</script>. La instrucción siguiente, dentro de la función CambiarFuente(), cambia el valor de la propiedad font-size de CSS. Si la propiedad está definida su valor será reemplazado y si no está definida, el valor se le asignará directamente.

Uso de CSS desde JavaScript

    El uso conjunto de CSS con JavaScript permite implementar diversos tipos de reacciones a determinados eventos que se producen en la interfaz (por ejemplo onClick, onMouseOut, onMouseOver, etc.).

    En el original se muestran ejemplos de determinadas reacciones a determinados eventos, como son:
  • Ampliación de texto al pasar el ratón sobre el mismo.
  • Ocultar y mostrar texto.
  • Zoom de imágenes.

Pseudo clases

    Utilizando CSS con JavaScript se pueden conseguir estilos dinámicos. Una alternativa sin utilizar scripts es el uso de pseudo clases. Las pseudo clases básicas son cuatro (aunque en CSS3 se incluyen algunas más):
  • :link.- define el estilo de un enlace que no ha sido visitado.
  • :visited.- ... enlace visitado.
  • :active.- ... que está siendo pulsado.
  • :hover.- sobre el que se pasa el ratón.
