Manejo de clases CSS

    A continuación se modificarán los elementos de una página web añadiendo y quitando clases CSS. Estas operaciones son simples porque en jQuery los elementos tienen dos métodos llamados addClass() y removeClass() que permiten aplicar o eliminar clases CSS.

    En el siguiente ejemplo existe un DIV con un texto y, fuera del DIV, un enlace. Al situar el ratón sobre el enlace se le añadirá una clase CSS al DIV y al retirar el ratón se eliminará la misma. El funcionamiento puede verse pinchando aquí.

    El código que interviene es el siguiente:

. . .
<style type="text/css">
    .verde {background-color: green;}
</style>
<script>
    $(document).ready(function() {
        $("a").mouseover(function(evento) {
            $("#texto").addClass("verde");
        })
        $("a").mouseout(function(evento) {
            $("#texto").removeClass("verde");
        })
    })
</script>
</head>
<body>
  <div id="texto">
Este texto se encuentra dentro de una capa a la que se añadirá
una clase CSS cuando el ratón se encuentre sobre en enlace de debajo.
La misma clase se eliminará al sacar el ratón del enlace.
  </div>
  <br>
  <a href="http://www.google.es">Al situar el ratón encima
     de este enlace se añade una clase a la capa superior</a>
. . .

    Hay que destacar:
  • La forma de referenciar a las etiquetas "a".
  • La forma de referenciar por id con el carácter "#".
  • Los nombres de los eventos.
  • El uso del método ready para operar solamente cuando la estructura de la página está cargada.

Manejo de propiedades CSS

    Para modificar los valores de las propiedades CSS, jQuery dispone del método css() que se podrá invocar sobre los elementos de la página que se deseen modificar.

    En el siguiente ejemplo va a modificarse el valor de la propiedad display para ocultar ("none") o mostrar ("block") los elementos.

Vamos a ver brevemente cómo usar el mencionado método css(). Primero, tendríamos que tener un elemento en la página, que es el que vamos a mostrar u ocultar.

<div id="mielemento">
Contenido del elemento...
</div>

Para ocultar/mostrar este elemento, habría que invocar el método css() utilizando una de las siguientes formas:

$("#mielemento").css("display", "none");
$("#mielemento").css("display", "block");

    Con el método css() pueden alterarse varias propiedades a la vez utilizando la sintaxis:

$("selector").css("propiedad1":"valor1"[,"propiedadn":"valorn"][...]);

    Si css() recibe un solo parámetro de tipo string, devuelve el valor CSS asignado a ese parámetro.

    Como práctica puede  utilizarse un formulario  con un cuadro de verificación (checkbox) que al activarse muestre otros contenidos y al desactivarse se oculten. El código y el resultado podrían ser los que se muestran a continuación:

<html>
<head>
   <title>Mostrar Ocultar</title>
<script src="jquery-1.3.2.min.js" type="text/javascript"></script>   
<script>
$(document).ready(function(){
   $("#mayoria_edad").click(function(evento){
      if ($("#mayoria_edad").attr("checked")){
         $("#formulariomayores").css("display", "block");
      }else{
         $("#formulariomayores").css("display", "none");
      }
   });
});
</script>
</head>
<body>
<form>
Nombre: <input type="text" name="nombre">
<br>
<input type="checkbox" name="mayor_edad" value="1" id="mayoria_edad" checked> Soy mayor de edad
<br>
<div id="formulariomayores" style="display: block;">
Dato para mayores de edad: <input type="text" name="mayores_edad">
</div>
</form>
</body>
</html>
