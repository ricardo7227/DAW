Operaciones de conversión a número

    Las operaciones aritméticas se realizan sobre constantes y variables numéricas. Si una variable contiene números pero está almacenada como cadena, antes de operar con ella debe ser convertida a número. Para ello existen dos funciones:
  • parseFloat() .- convierte a un número con decimales.
  • parseInt() .- convierte a un número entero.
    La conversión de una cadena de caracteres a número se realiza carácter a carácter, desde la primera posición y hasta encontrar el primer carácter no convertible a número (un punto se acepta en conversión de tipo parseFloat()), momento en el que la conversión se da por terminada y se convierte a número la primera parte de la cadena.

    NaN

    Si la función de conversión no encuentra un número en el inicio de la cadena a convertir entonces la función devuelve NaN, que significa Not a Number, que es una constante de tipo numérico sin valor. La función isNaN() permite averiguar si una variable es numérica. Para ello analiza el contenido completo de la variable y devuelve false o true dependiendo de  si el contenido es o no un número. Hay que tener muy en cuenta que isNaN() devuelve true cuando el contenido de la variable no es un número. Ver ejemplos de pag. 456.

Operación inversa (de número a cadena)

    La función toString() convierte una variable numérica a tipo String. Un ejemplo es:

          var cadena = toString(3527);

    Un método indirecto sería:

          var cantidad = 3000;
          cantidad = cantidad + " euros";

Objeto Math

    El objeto Math permite realizar operaciones complejas a través de una diversidad de métodos:
  • ceil().- devuelve el entero superior más cercano al pasado como parámetro.
  • floor().- devuelve el entero inferior más cercano al pasado como parámetro.
  • max().- devuelve el valor mayor de los pasados como parámetros.
  • min().- devuelve el valor menor de los pasados como parámetros.
  • round().- devuelve el entero más cercano.
  • abs().- valor absouto.
  • acos().- el ángulo cuyo coseno corresponde al valor pasado.
  • asin().- ...
  • atan().- ...
  • cos().- coseno del ángulo pasado como parámetro.
  • sin().- ...
  • tan().- ...
  • exp().- devuelve el número e (2,7182...) elevado al parámetro.
  • log().- logaritmo neperiano.
  • pow().- primer parámetro elevado al segundo.
  • sqrt().- raiz cuadrada.
  • random().- valor aleatorio entre 0 (incluido) y 1 (excluido).
    Si una operación aritmética no se puede resolver devuelve NaN.

    División entre 0

    En las divisiones entre 0 no se produce un error sino que devuelve Infinity o -Infinity.
