Tablas

    Además de los elementos ya conocidos que componen una tabla tenemos: grupo de columnas (colgroup), encabezamiento de filas (thead), cuerpo de filas (tbody) y pié de fila (tfoot).

    Y como atributos de celda:
  • headers.- identifica el id de la celda de encabezamiento que corresponde a la actual.
  • char.- especifica el caracter tomado como referencia en la alineación, por lo general el punto decimal, cuando se especifica align="char".
  • charoff.- desplazamiento desde el caracter de alineación.
    Con la etiqueta th se puede utilizar el atributo scope para informar de a quién se aplica el encabezado; admite los valores:
  • row.- el encabezamiento se aplica a algunas de las celdas siguientes de la misma fila.
  • col.- el encabezamiento se aplica a algunas de las celdas siguientes de la misma columna.
  • rowgroup.- el encabezamiento se aplica al resto de celdas del grupo de filas (el elemento debe pertenecer a un grupo de filas).
  • colgroup.- el encabezamiento se aplica al resto de celdas del grupo de columnas (el elemento debe pertenecer a un grupo de columnas).
    Además de los atributos rowspan y colspan que permiten hacer agrupaciones de filas y columnas en una única celda, las etiquetas colgroup y col permiten definir la agrupación de columnas de un modo declarativo, justo después de la etiqueta table y antes de asignar los contenidos de las celdas.

    Pueden verse ejemplos de las etiquetas y atributos en el libro original.

    Los elementos thead, tfoot y tbody representan las filas de cabecera, pié y cuerpo y deben aparecer en este orden.

    Puede utilizarse una tabla para crear una barra de navegación.
