Conceptos básicos

    El libro utiliza como sinónimos etiqueta, elemento y directiva HTML.

    Si para un atributo se omite su valor podrán utilizarse indistintamente las sintaxis:

          atributo=""
          atributo

    Se recuerda que el valor del atributo puede incluirse entre comillas simples o dobles.

    La solución para los caracteres especiales es el uso de juegos de caracteres especializados que incluso pueden utilizar estándares de 16 bits. El modo de definir el juego de caracteres es mediante la etiqueta <meta charset=" . . . "> para indicar el estándard elegido, por ejemplo, UNICODE (UTF-8).

    Para superar los problemas de algún caracter puntual puede utilizarse la codificación ya conocida. Por ejemplo la letra ñ puede codificarse de las siguientes formas:

    Formato decimal: &#241
    Formato hexadecimal: &#F1
    Por nombre: &ntilde;

    El apéndice B contiene tablas con las diferentes codificaciones.

    Atributos comunes a todas las etiquetas:
  • accesskey.- permite definir un acceso directo de teclado para que el elemento reciba el foco.
  • class.- permite definir una clase a la etiqueta que después se puede asociar a una clase CSS.
  • contextmenu.- permite asociar un menú de contexto que deberá estar definido en una etiqueta MENU.
  • contenteditable.- valor booleano que indica si el contenido delimitado por la etiqueta se puede modificar.
  • dir.- indica la direccionabilidad del elemento.
  • draggable.- indica si el elemento puede ser arrastrado (y soltado).
  • hidden.- elemento oculto que podrá visualizarse con código JavaScript.
  • id.- identificador. No admite espacios.
  • lang.- idioma del elemento.
  • spellcheck.- indica si se comprueba la gramática y ortografía.
  • style.- estilo.
  • tabindex.- orden de tabulación.
  • title.- microayuda para el elemento.
  • eventos.- puede asociarse una larga lista de eventos para ejecutar un script relacionado, por ejemplo: onclick, ondblclick, onmouseup, onmouseover, onfocus, onplay, onreset, etc.
    Etiqueta <html>

    Etiqueta <head>.- contiene las etiquetas title, base, link, meta e isindex.

    Etiqueta <title>.- los programas buscadores asignan a esta etiqueta un valor muy alto.

    Etiqueta <base>.- permite especificar el directorio base para las referencias relativas. Admite los atributos target, para especificar un nombre de destino válido para hiperenlaces y formularios, y href o dirección URL válida que se utilizará como dirección base. Muy utilizada para probar las páginas
en una ubicación diferente a la definitiva.

    Etiqueta <link>.- permite establecer enlaces con otros recursos. Admite como atributos:

       rel.- indica el tipo de enlace. Para hojas de estilo el valor es stylesheet.
       href.- especifica la dirección del documento enlazado.
       hreflang.- lenguaje del recurso.
       title.- para asignar un nombre a la relación.
       type.- tipo de documento.
       media.- tipo de dispositivo.

    Etiqueta <meta>.- es un contenedor de metainformación relacionada con el documento. Debe tener siempre uno de estos tres atributos: name, charset o http-equiv. Sus atributos más usados son:

       name.- identifica el tipo de información, por ejemplo: application-name, keywords, description, author, generator y robots.
       charset.- conjunto de caracteres del documento.
       content.- indica los metadatos. Depende del tipo de etiqueta meta.
       http-equiv.- para indicar alguna directiva pragma: content-languaje, content-type, default-style,refresh, set-cookie, prefetch, expires y no-cache.

    Etiqueta <style>.- permite incluir información sobre el estilo dentro del propio documento. Admite los atributos:

       type.- indica el lenguaje de estilo. Por defecto se presupone "test/css".
       media.- indica a qué medio se aplica. Por defecto es "all".
       title.- define una hoja de estilo alternativa.

    Etiqueta <script>.- usada para delimitar código de JavaScript. Admite como atributos:

       charset.- conjunto de caracteres del script.
       src.- dirección donde se encuentra el script.
       type.- lenguaje utilizado en el script. Por defecto es Javascript.
       async.- indica si el script se ejecuta inmediatamente.
       defer.- indica que el script se ejecuta cuando se termine la carga de la página.
