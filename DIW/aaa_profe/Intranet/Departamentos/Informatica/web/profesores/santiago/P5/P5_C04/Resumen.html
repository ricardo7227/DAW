Instalación de MySQL y PHPMyAdmin

    La instalación de MySQL se describe en el capítulo 11 de la Guia de Ubuntu Server.

    Para comprobar si el servidor de MySQL se está ejecutando puede teclearse como usuario root la instrucción:

netstat -tap | grep mysql

    Si está en ejecución se mostrará una línea similar a la siguiente:

tcp  0  0  localhost.localdo:mysql   *:*   ESCUCHAR

    Si no está en ejecución puede lanzarse con la instrucción:

/etc/init.d/mysql restart

    Para entrar al intérprete de mysql puede ejecutarse la instrucción:

mysql -u root -p

    A continuación pide la clave; por defecto durante la instalación de MySQL se da de alta al usuario root sin contraseña pero si no se conecta puede cambiarse la contraseña del root de MySQL con:

dpkg-reconfigure mysql-server-5.1

    La instrucción exit abandona el intérprete. El servicio se detiene con "/etc/init.d/mysql stop".

    Algunas instrucciones a teclear pueden ser:

SHOW DATABASES;
CREATE DATABASE banca;
SHOW TABLES;
CREATE TABLE banca.clientes (dni VARCHAR(8), nom VARCHAR(30), nac DATE, sal INT);
DESCRIBE banca.clientes;
LOAD DATA INFILE '/home/santiago/Mysql/clientes' INTO TABLE banca.clientes;
DELETE FROM banca.clientes;
INSERT INTO banca.clientes VALUES ('12345678', 'Pedro', '1963-10-12', 50000);
SELECT * FROM banca.clientes;
SELECT * FROM banca.clientes WHERE nac > '1970-10-23';
UPDATE banca.clientes SET nac = '1960-10-30' WHERE dni = '41111111';
SELECT nom, nac FROM banca.clientes;
SELECT DISTINC nom FROM banca.clientes;
ORDER nom, sal FROM banca.clientes ORDER BY sal [DESC], nac [DESC];
SELECT COUNT(*) FROM banca.clientes WHERE sal > 1500;
DROP DATABASE banca;

    Por defecto las bases de datos se crean en /var/lib/mysql

    La instalación de PHPMyAdmin se describe en el capítulo 12 de la Guia de Ubuntu Server.

    Debe comprobarse si se encuentra instalado el paquete phpmyadmin. Durante su instalación se solicita la siguiente información:
  • Servidor web a configurar: Apache2.
  • Desea configurar la base de datos para phpmyadmin con dbconfig-common ? SI (Activado).
  • Contraseña del usuario de administración de la base de datos: la de root.
  • Contraseña de aplicación MySQL para phpmyadmin: la de root.
  • Paso siguiente para la instalación de la base de datos: ignorar.
    Una vez instalado se ejecuta desde el navegador mediante la dirección "http://localhost/phpmyadmin".

     Dos de los ficheros de configuración más importantes son:
  • /etc/phpmyadmin/config.inc.php
  • /etc/phpmyadmin/apache.conf
    Si durante la ejecución aparece el error: "Respuesta: la conexión para controluser, como está definida en su configuración, fracasó", puede solucionarse modificando en el fichero /etc/phpmyadmin/config.inc.php las dos líneas siguientes:

           $cfg['Servers'][$i]['controluser'] = $dbuser;
           $cfg['Servers'][$i]['controlpass'] = $dbpass;

    Para que queden de la forma:

            $cfg['Servers'][$i]['controluser'] = '';   // Comilla simple repetida
            $cfg['Servers'][$i]['controlpass'] = '';   // Comilla simple repetida

PHPMyAdmin: una herramienta de administración

    PHPMyAdmin es una herramienta desarrollada en PHP que permite administrar MySQL dentro del entorno web. Con esta herramienta se puede:
  • Crear y eliminar bases de datos.
  • Crear, modificar y eliminar tablas.
  • Eliminar, modificar o añadir campos a una tabla.
  • Ejecutar sentencias SQL.
  • Administrar y crear índices y privilegios.
  • Administrar usuarios y múltiples servidores.
  • Cargar tablas a partir de archivos de texto.
  • Exportar datos a distintos formatos.
    PHPMyAdmin es código abierto cuya estructura sigue las líneas generales básicas:
  • Todos los datos de configuración se encuentran en /etc/phpmyadmin/config.inc.php.
  • Todos los archivos PHP incluyen tres archivos de inclusión: header.inc.php (el diseño), libraries/common.lib.php (funciones más comunes) y config.inc.php (archivo de configuración).
    Usuarios de PHPMyAdmin

    Para iniciar una sesión se pone en el navegador la dirección http://localhost/phpmyadmin y se introduce el usuario y contraseña solicitados, que será validado por el servidor MySQL, en el que deberá estar dado de alta.

    En la ventana principal se muestra la versión de PHPMyAdmin, la del servidor MySQL, la del cliente, el usuario que ha iniciado la sesión ,etc.

    Una descripción de la herramienta puede verse a partir de la página 992.

    Es interesante observar que cuando se realiza cualquier operación se muestra la sentencia SQL que se ejecuta.
