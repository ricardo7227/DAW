Propiedad content

    La propiedad content suministra el contenido utilizado por alguno de los pseudo elementos (:before y :after). Aunque su uso no está limitado, según el estándar debería utilizarse dentro de los pseudo elementos citados. Esta propiedad se aparta del esquema que propone que CSS se encarga del estilo y se desentiende del contenido; al ser una propiedad que genera contenido es un caso atípico dentro de CSS. Su formato es muy simple:

          content: "contenido";

    El contenido debe ser texto simple, y así se tomará aunque incluya, por ejemplo, etiquetas HTML. El texto se mostrará con los atributos heredados de su elemento padre. Una alternativa de formato es la siguiente:

          content: url('dirección URL de un soporte media');

    Así, dos ejemplos válidos serían:

          p:before {content: "Contenido generado automáticamente";}
          div.clase:after {content: url('audio.wav');}

Pseudo elementos

    Los pseudo elementos permiten asignar propiedades para áreas especiales de un documento con límites diferentes a los estándar. Los pseudo elementos disponibles son:
  • :first-line.- primera línea.
  • :first-letter.- primera letra.
  • :before.- antes del elemento.
  • :after.- después del elemento.
    El pseudo elemento se coloca como un sufijo del selector. En CSS3 los pseudo elementos comenzarán con dos puntos (:), dado que la codificación con un punto (.) se destina a las pseudo clases.

    Cuando dentro del contenido de la propiedad content quieran incluirse las dobles comillas como parte de la cadena a insertar, deberá utilizarse el objeto open-quote y close-quote en la forma:

          p {content: "El libro se titula " open-quote "Mio Cid" close-quote " y tiene 1.000 páginas";}

    Dentro de la propiedad content puede incluirse el objeto counter para numerar automáticamente una lista de elementos. Con counter se gestionan contadores que podrán reiniciarse o incrementarse automáticamente. El formato del contador es el siguiente:

          content:  "Texto opcional" counter (nombre_variable) " : ";

    Para que el contador se incremente automáticamente se utiliza la propiedad counter-increment que tiene el formato:

          counter-increment:  nombre_variable  valor_incremento;

    El valor de incremento predeterminado es 1. Veamos un ejemplo:

          counter-increment: contador  2;

    Para que el contador se reinicie se utiliza la propiedad counter-reset, que tiene la forma:

          counter-reset:  nombre_variable  valor_reinicializacion;

    El valor de reinicialización predeterminado es 1.

    En una misma propiedad counter-increment o counter-reset pueden incrementarse/reiniciarse varias variables. Un ejemplo:

          counter-increment: vara 2  varb 3;

    Mediante el objeto counter pueden numerarse listas, personalizando los valores de inicio, incremento, etc. Un ejemplo puede ser el siguiente:
