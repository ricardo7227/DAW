DOM: Modelo de objeto Documento

    El modelo de objeto documento, DOM, es un estandard para representar el documento independientemente del navegador que se utilice. Esto permite acceder a un documento mediante un conjunto de objetos, propiedades, métodos y eventos, lo que facilita la modificación dinámica del contenido de una página web mediante código de script.

    El estandar DOM ha tenido varias fases de desarrollo, lo que ha dado lugar a distintos niveles, siendo en estos momentos el 2 el nivel más aceptado.

DOM o BOM

    El modelo de objeto de navegador, BOM, se suele confundir con DOM pero no es lo mismo. Mientras DOM se centra en el documento, el modelo BOM incluye acceso a todas las áreas del navegador. Además, BOM es bastante específico de cada navegador mientras que DOM pretende ser totalmente independiente del navegador.

Objetos DOM

    La estructura de un documento se basa en que existe un nodo de nivel superior denominado nodo raiz debajo del que se desarrollan los nodos dependientes. Dada esta estructura, DOM necesita definir tres objetos base:
  • Nodo.- cada objeto del documento se define en un objeto Node. Existen distintos tipos de nodos.
  • NodeList.- es la lista de objetos Node.
  • NamedNodeMap.- este objeto permite acceder a todos los objetos Node utilizando el nombre en lugar de un índice.
    Tipos de nodos

    Todos estos objetos son nodos que heredan las propiedades y métodos del objeto Node pero la mayor parte de ellos además poseen métodos y propiedades propios:
  • Document.- nodo raiz del documento.
  • DocumentType.- DTD o esquema del documento.
  • DocumentFragment.- almacenamiento temporal para secciones del documento.
  • EntityReference.- una referencia a una entidad del documento.
  • Element.- un elemento del documento.
  • Attr.- un atributo de un elemento del documento.
  • ProcessingInstruction.- una instrucción de procesamiento.
  • Comment.- un comentario en el documento.
  • Text.- texto que forma un nodo hijo de un elemento.
  • CDATASection.- sección de CDATA dentro del documento.
  • Entity.- una entidad en el DTD.
  • Notation.- una notación definida en el DTD.

Node

    El objeto Node contiene métodos y atributos que se pueden utilizar para todos los tipos de nodos.

    Propiedades de Node
  • attributes.
  • childNodes.
  • firstChild.
  • lastChild.
  • nextSibling.
  • nodeName.
  • nodeType.
  • nodeValue.
  • ownerDocument.
  • parentNode.
  • previousSibling.
  • Etc.
    Principales métodos de Node
  • appendChild(n).- añade n al final de la lista y lo devuelve.
  • cloneNode(n).- devuelve un duplicado de n.
  • hasChildNodes().- retorna un valor booleano dependiendo de si el nodo tiene nodos hijos.
  • insertBefore(n1, n2).- inserta n1 antes de n2.
  • removeChild(n).- elimina n.
  • replaceChild(n1, n2).- reemplaza n2 por n1.

NodeList

    El objeto NodeList contiene un grupo ordenado de nodos a los que se accede por un índice. Sus propiedades son:
  • item(i).- devuelve el nodo i de la lista.
  • length.- cantidad de nodos de la lista.

NamedNodeMap

    El objeto NamedNodeMap representa una colección no ordenada de nodos que se recuperan por nombre.

Document

    El tipo de nodo Document representa a todo el documento. Es el nodo raíz del árbol jerárquico DOM.

    Propiedades del nodo Document
  • doctyp.
  • documentElement.- elemento raiz de ESTE documento.
    Métodos del nodo Document
  • createAttribute(atr).- crea el atributo atr.
  • createElement(etiq).- crea el elemento etiq.
  • createTextNode(texto).- crea un nodo de tipo Text con el texto correspondiente.
  • getElementById(id).- devuelve el elemento con identificador id.
  • getElementsByTagName(etiq).- devuelve un NodeList con los elementos del documento que tienen como etiqueta etiq.
  • Etc.

Element

    El tipo de nodo Element amplía el objeto Node con varios métodos y propiedades.

    Propiedades del nodo Element
  • tagname.
    Métodos del nodo Element
  • getAttribute(atr).- devuelve el valor del atributo atr.
  • getAttributeNode(atr).- devuelve un nodo que contiene el atributo atr.
  • getElementsByTagName(etiq).- devuelve un objeto NodeList con los descendientes de etiq.
  • removeAttribute(atr).- elimina el atributo atr.
  • removeAttributeNode(atr).- elimina el atributo atr y lo devuelve.
  • setAttribute(atributo, valor).- asigna valor a atributo.
  • setAttributeNode(atr).- añade un nodo atributo.
