Aplicación de estilo en selectores
Los estilos se pueden aplicar sobre toda directiva HTML que tenga representación visual y, para cumplir esos fines, pueda tener atributos de estilo asociados. Hemos visto cómo aplicar un estilo a un selector determinado, por ejemplo:
p {background: blue}
Esto implica que todos los párrafos tendrán fondo azul. Muchas veces no queremos que todos los selectores de un tipo determinado tengan el mismo estilo y esto se puede ajustar mediante la definición de clases a nivel de selector, por ejemplo:
p.resaltado {background: blue}
p.normal {background: yellow}
La solución es simple, se define el selector y se define un nombre especial después del punto (por ejemplo, resaltado). Luego, para utilizar este estilo en el documento HTML se utiliza el atributo class dentro de la etiqueta <P>, por ejemplo:
<p class=resaltado>Esto saldrá en fondo azul.</p>
< class=normal>Esto saldrá en fondo amarillo.</p>

El esquema de la sintaxis es el siguiente:
selector.nombreclase {propiedad: valor;...
propiedad:valor}
