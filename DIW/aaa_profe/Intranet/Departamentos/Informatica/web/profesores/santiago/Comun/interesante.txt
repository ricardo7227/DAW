La visibilidad de un elemento se controla con visibility y con display (Pág. 330).
	<a href="javascript:void()" onmouseover="ocultar()">Pulse para ocultar</a>
	
	function ocultar(){
		object.style.display="none";
	}
	
	Ver el siguiente código completo:
	
		<html>
		<head>
		<script type="text/javascript">
		function demoDisplay()
		{
		document.getElementById("p1").style.display="none";
		}

		function demoVisibility()
		{
		document.getElementById("p2").style.visibility="hidden";
		}
		</script>
		</head>
		<body>

		<p id="p1">This is some text.</p>
		<p id="p2">This is some text.</p>

		<input type="button" onclick="demoDisplay()" value="Hide text with display property" />
		<input type="button" onclick="demoVisibility()" value="Hide text with visibility property" />

		</body>
		</html>
