Definición y creación de objetos en JavaScript

    JavaScript es un lenguaje que sabe trabajar con objetos. Un objeto es una entidad que reune datos (propiedades) y funciones (métodos) para gestionar esos datos.

    El tipo del objeto define su estructura (sus propiedades y métodos) y dentro de un programa pueden existir cero, una, o más instancias (ejemplares) de un objeto. Los objetos se definen en cualquier lugar del programa pero la creación física del programa se puede postergar hasta el momento en que realmente sea necesario.

    El tipo del objeto se denomina clase del objeto.

    Como ejemplo puede suponerse un objeto de tipo Ventana, podría tener las propiedades:
  • Alto.
  • Ancho.
  • Estado (abierta o cerrada).
    Y sus métodos podrían ser:
  • Abrir.
  • Cerrar.
  • Comprobar_estado.
    Las propiedades sirven para almacenar valores del objeto y los métodos para definir las acciones que puede realizar el objeto. Los objetos pueden ser definidos por el usuario o pueden ser objetos predefinidos de JavaScript, en ambos casos el modo de gestionar métodos y propiedades de un objeto es exactamente el mismo.

Objetos propios de JavaScript

    JavaScript contiene un gran número de objetos predefinidos o intrínsecos del lenguaje: el documento, las tablas, los formularios, el navegador, etc. Por ejemplo, todos los datos contenidos dentro de una página son propiedades del objeto "document", que tiene la propiedad "title" que almacena la cadena de caracteres del título de la página.

    Acceso a las propiedades de los objetos

    Algunas propiedades son de lectura/escritura y otras de solo lectura por lo que no se podrá modificar su valor. El acceso a la propiedad tiene la sintaxis:

            objeto.propiedad

    Por ejemplo:

            document.title = "Título de mi página"

    Aunque la sintaxis completa sería window.document.title = "..." dado que document es un objeto hijo del objeto window. JavaScript permite obviar la referencia al objeto window dado que la presupone automáticamente; si se expecifica es perfectamente correcto.

    Por último decir que una propiedad puede ser de cualquier tipo, incluido de tipo objeto.

    Uso de métodos de un objeto

    Los métodos son funciones asociadas a un objeto que realizan acciones relacionadas con el objeto. Algunos ejemplos de uso de métodos que ya se han visto anteriormente serían:

          window.document.write ("Hola JavaScript");
          window.alert ("Atención");

    En ambos casos no sería necesario hacer referencia al objeto window. Sencillamente, si se utiliza "alert ("Atención");" parece una función normal aunque en realidad es un método del objeto window; desde el punto de vista funcional es exactamente lo mismo.

Sentencia for ... in para objetos

    Para recorrer todas las propiedades de un objeto se puede utilizar el código:

          for  (var miPropiedad in miObjeto) {
          }

    Esto provoca un bucle por todas las propiedades del objeto miObjeto.

    Por ejemplo, pueden recuperarse todas las propiedades del objeto document:

          for  (var miPropiedad in document) {
                document.write (miPropiedad, " = ", document[miPropiedad], "<br>");
          }

Sentencia with

    La sentencia with simplifica la codificación pues permite establecer  un objeto como referencia de forma que para hacer uso de sus propiedades y métodos no es necesario especificar el nombre del objeto. Veamos un ejemplo:

          document.write ("El título es: ", document.title);
          document.write ("<br>");

    puede escribirse de la forma:

          with (document) {
                      write ("El título es: ", title);
                      write ("<br>");
          }

Definición y creación de objetos en JavaScript

    La definición de objetos personalizados se realiza de la forma siguiente:

    <!DOCTYPE html>
    <html>
    <head>
      <title>Mi primer Objeto</title>
      <script languaje="javascript">
      // Definición de un objeto
         function persona (txtNombre, txtApellido1, txtApellido2, nEdad) {
                    // Definición de propiedades (valores)
                        this.nombre = txtNombre;
                        this.apellido1 = txtApellido1;
                        this.apellido2 = txtApellido2;
                        this.edad = nEdad;
                    // Definición de métodos (funciones)
                        this.mostrar = fnVisualizar;
         }
      // Función relativa al método
         function fnVisualizar ()  {
                        with (this) {
                        document.write ("---------------------------------------------------", "<br>");
                        document.write ("|  Nombre.- <strong>", nombre, "</strong>  **<br>");
                        document.write ("|  Primer apellido.- <strong>", apellido1, "</strong>  **<br>");
                        document.write ("|  Segundo apellido.- <strong>", apellido2, "</strong>  **<br>");
                        document.write ("|  Edad.- <strong>", edad, "</strong>  **<br>");
                        document.write ("---------------------------------------------------", "<br>");
                        }
         }
      </script>
    </head>
    <body>
      <script type="text/javascript">
        var objProfe = new persona ( "Santiago", "Alonso", "Gomez", 48 );
        objProfe.mostrar ();
      </script>
    </body>
    </html>

    Definición de un método

    La definición de un método consiste en asignarle el nombre de una función que posteriormente se desarrolla.

    Debe observarse el uso de la palabra clave this; al estar dentro del objeto persona, this significa el objeto persona vigente.

    Puntero this para referenciar un objeto

    La palabra clave this significa el propio objeto que se está gestionando en ese código.

    Se define el objeto personalizado y se crea su método constructor, dentro del cual la palabra clave this se refiere al objeto que se está creando o manipulando en ese momento.

    Operador new para crear un objeto

    Para crear un objeto se utiliza el operador new que crea una copia del objeto. Los parámetros de la función se utilizan para asignar valores a las propiedades.

    En el ejemplo del principio de esta sección, la variable objProfe es una variable más dentro del programa, pero al haberla creado con new se crea un objeto de la clase persona con el nombre objProfe. Para acceder a sus propiedades y métodos se hace como en los ejemplos siguientes:

          objProfe.apellido2 = "Pérez";
          var iedad = objProfe.edad;
          objProfe.mostrar ();
          document.write (objProfe.nombre);

    Contención de objetos

    Las propiedades de un objeto pueden ser de cualquier tipo de dato stándar (numérico, booleano, cadena, fecha, etc.) y también de cualquier tipo de objeto intrínseco del lenguaje (Array, document, etc.) o creado por el usuario. Ver ejemplo de la pag. 414.

    La capacidad de que unos objetos se incluyan dentro de otros es una de las características de la programación orientada a objetos. Mediante la notación de puntos se puede hacer referencia a varios niveles de profundidad de la jerarquía de contención de objetos dejando perfectamente clara la dependencia entre los objetos.

    La sintaxis es la siguiente:

          objPadre.objHijo.propiedad = " valor ";
          objPadre.objHijo.metodo ();

    Copia de objetos

    Un objeto se puede crear por copia de otro existente pues nada impide la existencia de varias variables de un mismo objeto. La copia se hace de la forma:

          var objAlumno = objProfe;

    De esta forma se crea la variable objAlumno que es un objeto cuyo contenido coincide con el de objProfe, con la particularidad que ambos comparten las mismas direcciones de memoria y si se modifica uno de los dos objetos el otro también quedará modificado.

    Si se quiere copiar un objeto y que se puedan modificar de forma independiente deberá crearse con new y asignársele las propiedades del otro objeto de forma explícita. Por ejemplo:

          var objProfe  = new persona ("Santiago", "Alonso", "Gomez", 48);
          var objAlumno = new persona (objProfe.nombre, objProfe.apellido1, objProfe.apellido2, objProfe.edad);

    Para simplificar la tarea de copia los objetos suelen implementar un método como el siguiente:

          function copiar (objPersona) {
                   objPersona.nombre    = this.nombre;
                   objPersona.apellido1 = this.apellido1;
                   objPersona.apellido2 = this.apellido2;
                   objPersona.edad      = this.edad;
          }

    Y, para copiar un objeto haciendo uso de la función anterior sería:

          var objProfe = new persona ("Santiago", "Alonso", "Gomez", 48);
          objProfe.copiar(objAlumno);
