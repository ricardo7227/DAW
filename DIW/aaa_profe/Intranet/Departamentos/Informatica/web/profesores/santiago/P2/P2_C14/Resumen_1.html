    En las interfaces de usuario se utilizan una serie de elementos, como iconos, colores y fuentes, que son propios del sistema y que poseen características que se definen a nivel de la plataforma al configurar el sistema, por ejemplo, los colores asignados a cada parte de la interfaz de usuario y el tipo de letra que se usa en cada área. Desde CSS pueden aprovecharse esas definiciones de propiedades en la presentación de las páginas web.

Cursores

    La propiedad cursor permite establecer qué tipo de cursor se debería utilizar cuando el cursor del ratón está encima de un elemento específico de la página web. Su formato es:

          cursor: valor;

    El valor puede ser uno de los valores simbólicos que se detallan más adelante o la dirección url de una imagen personalizada de extensión .cur. En este último caso el formato queda:

          cursor: url("fichero.cur");

    La propiedad puede tener múltiples valores. Siempre tiene prioridad el primero de la lista, pero si no está disponible se utiliza el siguiente. Por ejemplo:

          cursor: url("primero.cur"),   url("segundo.cur");

    Los valores estándard de la propiedad cursor son:
  • auto.- a elección del agente de usuario.
  • crosshair.- el cursor toma la forma de un signo + de tamaño más grande.
  • default.- cursor predeterminado de la plataforma (generalmente el puntero de la flecha).
  • help.- el cursor que indica que existe una ayuda (una interrogación grande).
  • move.- forma de cuatro flechas para indicar que el elemento se puede mover.
  • pointer.- una mano apuntando.
  • progress.- cursor que indica una operación en marcha (un reloj de arena).
  • text.- cursor que se utiliza para indicar que se puede introducir texto.
  • wait.- puntero de espera (reloj de arena).
  • x-resize.- siendo x una de las combinaciones: n, s, e, w, ne, se, nw o sw. Se utiliza para indicar que el elemento puede ser redimensionado. La letra indica el sentido del punto cardinal.
  • url.- dirección de un recurso de tipo cursor.

Colores de la interfaz

    Además de las formas vistas de asignar colores a los elementos, existen los siguientes valores simbólicos, que se especifican tal cual aparecen a continuación:
  • ActiveBorder.- color del borde de la ventana activa.
  • ActiveCaption.- color del texto del título de la ventana activa.
  • AppWorkspace.- color de fondo de un interfaz de documento múltiple.
  • Background.- color del fondo del escritorio.
  • ButtonFace.- color de la cara del botón.
  • ButtonHighlight.- color del área de sombra oscura del borde de los botones 3D.
  • ButtonShadow.- color de la sombra del borde de los botones 3D.
  • ButtonText.- color del texto del botón.
  • CaptionText.- color del texto en títulos y cuadros.
  • GrayText.- color del texto no habilitado.
  • Highlight.- color que indica que el elemento está seleccionado.
  • HighlightText.- color que indica que el texto está seleccionado.
