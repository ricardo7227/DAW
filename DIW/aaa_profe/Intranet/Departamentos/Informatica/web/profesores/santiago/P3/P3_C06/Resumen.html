Gestión de cadenas de texto

    Para declarar una cadena de caracteres se utilizan las sintaxis:

          var variable = "cadena";   // Con comillas dobles
          var variable = 'cadena';   // Con comillas simples

    Si la cadena contiene comillas, estas deberán ir precedidas de un caracter de escape (\), por ejemplo:

          var miCadena = "El titulo del libro es \"JavaScript\"";

    Para unir o concatenar dos o más cadenas se utiliza el operador "+". Si el contenido de alguna de las cadenas es numérico se transforma temporalmente en cadena para poder realizar la concatenación. Por ejemplo:

          var miCad1 = "El titulo del libro es: ";
          var miCad2 = "\"JavaScript\"";
          var miCad  = miCad1 + miCad2;

Objeto String

    Las cadenas de caracteres son objetos de tipo String, con sus métodos y propiedades que facilitan su manipulación. Por ejemplo, la propiedad length contiene la longitud en bytes de la cadena. Una característica es que para su creación no es necesario utilizar el operador new.

    El acceso a los métodos y propiedades de un objeto String se hace mediante la notación de puntos conocida. Por ejemplo, miCadena.length contiene la longitud de miCadena.

    Una cadena se puede expresar mediante su nombre de variable (por ejemplo, miCadena) o directamente por la propia cadena de caracteres, lo que permite una codificación como la siguiente:

          var intLongitud = "El titulo del libro es: \"JavaScript\"".length;

    Métodos más comunes

  • charAt().- accede a un carácter de la cadena, utilizando un índice que comienza por 0.
  • charCodeAt().- devuelve el valor numérico decimal del código ASCII correspondiente a un carácter de la cadena.
  • fromCharCode().- devuelve los caracteres ASCII que corresponden a los códigos decimales (separados por comas).
  • substring().- devuelve una subcadena de una cadena a partir de una posición de inicio y fin.
  • slice().- es equivalente al método substring().
  • substr().- devuelve una subcadena de una cadena a partir de una posición de inicio y una longitud.
  • indexOf().- devuelve la posición de inicio de una subcadena en una cadena. Si la subcadena no está contenida en la cadena el valor devuelto es -1.
  • lastIndexOf().- devuelve la posición de inicio de la última subcadena en una cadena. Si no existe devuelve -1.
  • toLowerCase().- convierte a minúsculas.
  • toUpperCase().- convierte a mayúsculas.
    Para la codificación y decodificación de caracteres especiales existen las funciones escape() y unescape().

    Métodos vinculados al formateo HTML

    Existe un conjunto de métodos para el objeto String que está relacionado con las etiquetas HTML de formateo de texto. Por ejemplo:

          var miCadena = "Esta es la cadena";
          document.write (miCadena.bold());   // Dará como resultado <b>Esta es la cadena</b>

    Es decir, el mecanismo de las funciones que se describirán a continuación consiste en añadir etiquetas HTML a las cadenas de caracteres. Con este esquema existen los siguientes métodos:
  • anchor().- crea las etiquetas <a name="parámetro"> cadena </a>. Así, la instrucción "La cadena".anchor("elname") generaría el código: <a name="elname"> La cadena </a>
  • big().- genera <big> La cadena </big>.
  • blink().- genera <blink> La cadena </blink>.
  • bold().- genera <b> La cadena </b>.
  • fixed().- genera <tt> La cadena </tt>.
  • fontcolor(miColor).- genera <font color="miColor"> La cadena </font>.
  • fontsize(miSize).- genera <font size="miSize"> La cadena </font>.
  • italics().- genera <i> La cadena </i>.
  • link(url).- similar a anchor(). Genera <a href="url"> La cadena </a>. Por ejemplo, "Pincha aquí".link("www.tiernogalvan.es") generará <a href="www.tiernogalvan.es">Pincha aquí</a>.
  • small().- genera <small> La cadena </small>.
  • strike().- genera <strike> La cadena </strike>.
  • sup().- genera <sup> La cadena </sup>.
  • sub().- genera <sub> La cadena </sub>.
    Hay que tener en cuenta que aunque disponemos de estos métodos, la tendencia debe ser que todo lo referente a estilos se sitúe en ficheros CSS.

    Acceso a los caracteres de una cadena

    El método charAt() permite acceder a un caracter de una cadena. Para recorrer todos los caracteres de una cadena puede utilizarse el siguiente código:

          var miCadena = "Esta es la cadena";
          for (var i = 0; i < miCadena.length; i++) {
              document.write(miCadena.charAt(i));
          }

    El método charAt() es un método de lectura por lo que la siguiente instrucción sería incorrecta:

          miCadena.charAt(2) = "C";  // Es incorrecto.

    Código ASCII

    El método de lectura charCodeAt() devuelve el valor decimal correspondiente al caracter que ocupa la posición pasada en el parámetro (el primero es el 0) dentro de la cadena. Así, no es dificil conocer los códigos decimales de la tabla ASCII correspondientes a todos los caracteres de una cadena. Basta recorrer la cadena de forma similar al ejemplo anterior. Un ejemplo utilizando este método sería:

          document.write ("Esta es la cadena".charCodeAt(2));

    que en este caso concreto mostraría: 116.

    El método fromCharCode() permite definir una cadena a partir de los códigos decimales ASCII:

          var miCadena = String.fromCharCode(201, 115, 116, 97)
          // El valor asignado a miCadena es "Esta"

    Tabla de caracteres

    Para que el navegador sepa la tabla de caracteres que debe utilizar y el idioma utilizado en la página, puede añadirse el siguiente código:

          <?xml version="1.0" encoding="ISO-8859-1"?>
          <html xmlns="http://www.w3.org/1999/xhtml" xml:lang="es-ES" lang="es">
          . . .

    Subcadenas

    El método substring() permite extraer una subcadena de una cadena. Admite como parámetros la posición inicial y la final (que no se incluye en la extracción) de la subcadena. Por ejemplo:

          var miCadena = "Esta es la cadena";
          document.write (miCadena.substring(1,3));   // Dará como resultado st

    El método slice() es equivalente a substring().

    El método substr() es parecido; el segundo parámetro indica la longitud de la subcadena.

    Búsqueda dentro de una cadena

    El método lastIndexOf() devuelve el índice de la posición de la última subcadena coincidente con el texto buscado que aparece dentro de una cadena. Si no hay coincidencias devuelve -1. Un par de ejemplos:

          var str1 = "Texto1 Texto2".lastIndexof("Tex");   // Devuelve 7
          var str2 = "Texto1 Texto2".lastIndexOf("TTT");   // Devuelve -1

    Este método puede ser útil para obtener la extensión del nombre de un fichero.

    El método indexOf() es similar solo que devuelve la posición de la primera aparición de una subcadena dentro de la cadena. Opcionalmente se puede indicar un punto de partida de la búsqueda.

    Conversión a mayúsculas o minúsculas

    Los métodos toLowerCase() y toUpperCase() permiten pasar una cadena a minúsculas o mayúsculas respectivamente. Un ejemplo de uso puede ser para comparar dos cadenas sin que se diferencien las mayúsculas de las minúsculas. Estos métodos combinados con substring(), substr() y charAt() permiten cambiar subcadenas.

    Evaluación de una cadena

    La función eval() permite evaluar una cadena de caracteres, ejecutándo el código y devolviendo el resultado. Entre sus utilidades se encuentra la de ejecutar un código creado de manera dinámica dentro de una variable de tipo String. Veamos un ejemplo:

          var numpag = 10;
          document.write (eval("numpag"));  // Visualiza el contenido de numpag: 10
          var var1 = "numpag";
        // Modo indirecto (sin comillas)
          document.write (eval(var1));      // Visualiza el contenido de numpag: 10
          document.write (eval("20 + 4");   // Visualiza 24

    Escape de caracteres

    Algunos caracteres requieren el uso de caracteres de escape para no generar problemas en su utilización dentro de una página web. Algunos de los casos más usuales son las comillas, las vocales acentuadas, etc.

    La función escape() codifica la cadena de forma que los caracteres especiales quedan representados mediante el formato %H, siendo H el número hexadecimal correspondiente. Para recuperar una cadena codificada, a su formato original, se utiliza la función unescape(), que devuelve una cadena limpia con los caracteres "escapados" convertidos a su valor real. Un ejemplo sería:

          var miCadena1 = "áéíóú aeiou &%'";
          document.write (escape(miCadena1) + "<br>");  // Devuelve %E1%E9%ED%F3%FA%20aeiou%20%26%25%27
          var miCadena2 = unescape(miCadena1);
          document.write (miCadena2);                   // Devuelve áéíóú aeiou &%'

Expresiones regulares

    Las expresiones regulares simplifican ciertas manipulaciones de cadenas, verificaciones de formatos (e-mails, teléfonos, DNI's, ...), extracciones de subcadenas, etc. Veamos un ejemplo que valida una cadena de texto en la que solamente se aceptarán letras y espacios y cuya longitud no sobrepasará los 20 caracteres:

          <html>
          <head>
           <script type="text/javascript">
            function esVálido(str1){
                     // se crea la expresión regular
                     var expreg = new RegExp("^[a-zA-Z ]{1,20}$","g");
                     if (expreg.test(str1)){
                        return str1 + " es OK";
                     } else {
                        return str1 + " es KO";
                     }
            }
            // KO: por tener un número
            document.write(esVálido("1Ricardo"), "<BR />");
            // OK
            document.write(esVálido("Juan"), "<BR />");
            // KO: por tener un acento
            document.write(esVálido("José"), "<BR />");
            // OK
            document.write(esVálido("Carlos Alberto Salas"), "<BR />");
            // KO: demasiado largo
            document.write(esVálido("Carlos Alberto Salas Fuentes"), "<BR />");
           </script>
          </head>
          <body>
          </body>
          </html>

    Cuyo resultado es:

1Ricardo es KO
Juan es OK
José es KO
Carlos Alberto Salas es OK
Carlos Alberto Salas Fuentes es KO

    El uso de las expresiones regulares tiene el problema de dominar la definición de la cadena de texto que describe la validación deseada. Una vez conseguida, se utilizará el método test del objeto RegExp que devolverá verdadero o falso dependiendo de si la cadena en cuestión cumple con la cadena de verificación.

    Definición de expresiones regulares

    Para crear un objeto RegExp es preciso definir una expresión regular. Para crear una instancia al objeto se utiliza la sintaxis:

          var expreg = new RegExp (cadenaExpReg, opción);

    El parámetro cadenaExpReg es la cadena de texto de la expresión regular, que se analizará más adelante. El parámetro opción puede tener uno de estos valores:
  • g.- indica el proceso en toda la cadena.
  • i.- no diferencia entre mayúsculas y minúsculas.
  • gi.- combinación de las dos opciones anteriores.
  • "".- indica no aplicar ninguna de las opciones.
    También está permitida, para crear un objeto RegExp, la siguiente sintaxis:

          var expreg = /cadena_ExpReg/opción;

    Los caracteres que pueden utilizarse en una expresión regular son:
  • ^ .- inicio de cadena.
  • $ .- fin de cadena.
  • . .- cualquier carácter.
  • [xyz] .- grupo de caracteres permitido.
  • [a-z], [A-Z], [0-9] .- rango de caracteres válidos.
  • [^rango] .- rango de caracteres NO válido.
  • x|y .- El carácter x o el carácter y.
  • ( ) .- permiten agrupar caracteres.
  • * .- el carácter precedente se acepta de 0 a n veces.
  • + .- el carácter precedente se acepta de 1 a n veces.
  • ? .- el carácter precedente se acepta de 0 a 1 vez.
  • {n} .- el carácter precedente se acepta solo n veces.
  • {n, } .- el carácter precedente se acepta al menos n veces.
  • {n, m} .- el carácter precedente se acepta entre n y m veces.
  • \ .- carácter escape.
  • \b .- límite de palabra (espacio, salto de línea, inicio o fin de cadena).
  • \B .- cualquier carácter que no sea límite de palabra.
  • \d .- indica dígitos. Es similar a [0-9].
  • \D .- indica NO dígitos. Es similar a [^0-9].
  • \s .- indica caracteres de espaciado (espacio, tabulador o salto de página).
  • \S .- indica cualquier carácter que no sea de espaciado.
  • \w .- indica cualquier carácter alfanumérico (incluye el guión bajo (_)).
  • \W .- cualquier carácter que no sea alfanumérico ni el guión bajo.

    El método test() del objeto RegExp

    El objeto RegExp tiene el método test() para verificar si una cadena pasada como parámetro cumple con la expresión regular; si cumple devuelve true y si no devuelve false.

    Un ejemplo de una función que compruebe si una cadena corresponde a un NIF menor de 80000000-X podría ser:

          function validarNIF (cadena) {
                   expresion = new RegExp ("^[0-7][0-9]{7}-[A-Z]$", "g");
                   valido = (expresion.test (cadena));
                   return (valido);
          }

    El método exec() del objeto RegExp

    El método exec() devuelve la primera coincidencia encontrada dentro de una cadena pasada como parámetro en base a la expresión regular utilizada. Permite así extraer cierta información de dentro de un texto: dnis, teléfonos, direcciones de correo, ...

    El método match() del objeto String

    El método match() es similar a exec() con la particularidad de que, en vez de devolver la primera aparición, devuelve una tabla con tantos elementos como datos existan en el texto que coincidan con la expresión regular correspondiente. El método match() utiliza como parámetro una expresión regular pues define automáticamente un objeto RegExp.

    Un ejemplo de uso sería:

          var strCadena = "Este es el texto a analizar. En concreto bla, bla, bla, ...";
          var misDNIs = strCadena.match("[0-7][0-9]{7}-[A-Z]", "g");
          document.write (misDNIs);

    El método split() del objeto String

    El método split() permite obtener una tabla con las subcadenas detectadas entre los separadores expecificados (sin incluirlos).

    Un ejemplo de uso sería:

        // Lista de cadenas con distintos separadores
        var miCadena = "HTML 5 - CSS 3, JavaScript; PHP - MySQL";
        // Mostrar la cadena
        document.write("CADENA COMPLETA<br><br>");
        document.write(miCadena, "<br>");
        // Crear la expresión regular
        var expresion = new RegExp("[-,;]+", "g");
        // Crear tabla con las cadenas incluidas entre los separadores
        var tabla = miCadena.split(expresion);
        // Mostrar los elementos de la tabla a razón de uno por cada línea
        if (tabla == null) {
            document.write("La tabla está vacía. Comprobar la expresión regular");
        } else {
            document.write("<br>Subcadenas encontradas:<br><br>");
            for (var i = 0; i < tabla.length; i++) {
               document.write(tabla[i], "<br>");
            };
        };

    El método replace() del objeto String

    El método replace() del objeto String permite reemplazar caracteres dentro de una cadena de texto. Funciona en colaboración con una expresión regular. Puede utilizarse para insertar ciertas etiquetas HTML en un texto. Ver ejemplos de pag. 450.
