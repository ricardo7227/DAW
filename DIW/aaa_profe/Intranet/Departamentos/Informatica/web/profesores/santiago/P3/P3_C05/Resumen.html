Objeto Date

    El objeto Date es un objeto predefinido que permite manipular fechas y horas. El almacenamiento de la información dentro del objeto cumple las premisas:
  • El año se representa con cuatro dígitos.
  • El mes es un índice de una tabla, cuyos valores van de 0 (enero) a 11 (diciembre).
  • El día es un valor entre 1 y 31.
  • El día de la semana es un índice de una tabla, cuyos valores van de 0 (domingo) a 6 (sábado).
  • La hora se representa por un valor entre 0 y 23.
  • Los minutos y los segundos se representan por un valor entre 0 y 59.
  • Los milisegundos se representan por un valor entre 0 y 999.
  • El huso horario utiliza la configuración regional del equipo del cliente. Se indica el desplazamiento en horas respecto al meridiano de Greenwich.
    Un objeto de tipo Date se crea, como es de esperar, de la forma:

          var dtFecha = new Date()

    Y así, la instrucción:

          document.write (dtFecha);

    producirá la salida:

    El constructor puede utilizar los parámetros:

          Date (año, mes, día, hora, minuto, segundo)

    lo que permite crear una fecha con instrucciones como las siguientes:

          // Fecha: 7 / 12 / 2011
          var dtFecha1 = new Date (2011, 11, 7);
          // Fecha: 23 / 3 / 2007    Hora: 23:12:45
          var dtFecha2 = new Date (2007, 3, 23, 23, 12, 45);

Métodos de lectura

  • getDate().- devuelve el día de la fecha (de 1 a 31).
  • getDay().- devuelve el número del día de la semana (de 0 a 6).
  • getFullYear().- devuelve el año en cuatro dígitos.
  • getHours().- Devuelve la hora (de 0 a 23).
  • getMilliseconds().- Devuelve los milisegundos (de 0 a 999).
  • getMinutes().- Devuelve los minutos (de 0 a 59).
  • getMonth().- Devuelve el número de mes ( de 0 a 11).
  • getSeconds().- Devuelve los segundos (de 0 a 59).
  • getTime().- Devuelve la cantidad de milisegundos transcurridos desde el 1 / 1 / 1970 hasta la fecha del objeto.
  • getTimezoneOffset().- Devuelve la cantidad de minutos de diferencia entre la hora local y la zona horaria de referencia.
  • getYear().- Devuelve el año con cuatro cifras, pero se recomienda el uso del método siguiente.
  • getFullYear().- Devuelve el mismo dato.
  • getUTC...().- Estos métodos obtienen la información de fecha/hora pero en el modo UTC (coordenadas universales): getUTCDate(), getUTCDay(), getUTCFullYear(), getUTCHours(), getUTCMilliseconds(), getUTCMinutes(), getUTCMonth(), getUTCSeconds().
    Uso de los métodos getDate(), getMonth() y getFullYear()

             var  miFecha = new Date();
             with (miFecha) {
                 document.write ("La fecha del día de hoy es: " + getDate() + "/" + getMonth() + "/" + getFullYear())
             }
          </script>

                 var h = getHours();
                 if (h < 10) { h = "0" + h };

Métodos de escritura

  • setFullYear().- Asigna el año con cuatro dígitos.
  • setMonth().- Asigna el número de mes (de 0 a 11).
  • setDate().- Asigna el número del día de la fecha.
  • setHours().- Asigna la hora (de 0 a 23).
  • setMinutes().- Asigna los minutos (de 0 a 59).
  • setSeconds().- Asigna los segundos (de 0 a 59).
  • setMilliseconds().- Asigna los milisegundos (de 0 a 999).
  • setTime().- Asigna la cantidad de milisegundos transcurridos desde el 1/1/1970.
    Uso del método setDate()

          <script languaje="javascript">

             with (miFecha) {
                 document.write ("Fecha antes de la modificación..: ");
                 document.write (getDate() + "/" + getMonth() + "/" + getFullYear() + "<br>");
               // Modificamos el día cambiandolo por el 8
                 setDate(8);
                 document.write ("Fecha despues de la modificación: ");
                 document.write (getDate() + "/" + getMonth() + "/" + getFullYear());
             }
          </script>

    Uso del método setTime()

          <script languaje="javascript">
             var miFecha = new Date();
           // Cantidad de milisegundos por día
             var msDia = 24 * 60 * 60 * 1000
             with (miFecha) {

                 setTime(getTime() + 90 * msDia);
                 document.write ("Dentro de 90 dias será: ");
                 document.write (getDate() + "/" + getMonth() + "/" + getFullYear());

          </script>

    Función setTimeout()

    La función setTimeout() permite definir una función que se ejecutará cuando pase un tiempo determinado (especificado en milisegundos). Su sintaxis es:

          timer = setTimeout("fn()", retraso);

    Esta función hace que se ejecute la función "fn()" cuando transcurra el tiempo determinado en la variable "retraso".

    En el siguiente ejemplo se define una función que actualiza el título de la página y la barra de estado de la ventana cada segundo:

          <script languaje="javascript">
             function TituloDinamico() {
                     var miFecha = new Date();
                     var hor = miFecha.getHours();
                     var min = miFecha.getMinutes();
                     var seg = miFecha.getSeconds();
                     document.title = "Hora actual: " + hor + ":" + min + ":" + seg;
                     window.status  = "Hora actual: " + hor + ":" + min + ":" + seg;
                     var miTimer = setTimeout ("TituloDinamico()", 1000);
             }
             TituloDinamico();
          </script>

    En cada ejecución de la función vuelve a establecerse setTimeout() pues de lo contrario solamente se ejecutaría una vez. Para desactivar la variable que controla el proceso iniciado por setTimeout() se utiliza la función clearTimeout(miTimer), lo que provoca que la función no vuelva a ser llamada.

    Función setInterval()

    La función setInterval() es muy similar a la anterior, con una única diferencia: la llamada dinámica a la función no se interrumpe nunca sino que se reinicia automáticamente cada vez que se cumple el intervalo. La única manera de interrumpir el proceso es mediante el uso de la función clearInterval().

    Supongamos una función que muestra el tiempo transcurrido en la página:

             function TiempoPasado() {

                     var tiempo = Math.round((FechaAct.getTime() - msIni) / 1000);
                     document.title = "Segundos transcurridos: " + tiempo;
             }
             var FechaIni = new Date();
             var msIni = FechaIni.getTime();
             TiempoPasado();
          // Actualiza el título con una llamada
             var miTimer = setInterval ("TiempoPasado()", 1000);
