Alineación horizontal (text-align)

La propiedad text-align de CSS puede tomar cuatro valores: left, right, center o justify. La alineación suele realizarse sobre párrafos pero también puede llevarse a efecto sobre celdas de tablas. Un ejemplo:

          p {text-align: justify}

Alineación vertical (vertical-align)

La propiedad vertical-align puede tomar los valores:

  • baseline.- valor predeterminado. El texto utiliza la línea base para alinearse respecto a otros objetos que lo rodean.
  • sub.- el texto desciende al nivel utilizado por los subíndices.
  • super.- ... por los superíndices.
  • top.- la parte superior de la línea o elemento se alinea con la parte superior de su elemento padre.
  • text-top.- la parte superior de la línea o elemento se alinea con la parte superior del texto de su elemento padre.
  • middle.- el medio del texto se alinea con la línea media de los elementos que lo rodean.
  • bottom.- ... la parte inferior del elemento padre.
  • text-bottom.- la parte inferior de la línea o elemento se alinea con la parte inferior del texto de su elemento padre.
  • length.- el elemento subirá (positivo) o bajará (negativo) según el valor asignado.
  • percentage.- el elemento subira o bajara un porcentaje relativo a la altura de la línea.

Sangría del texto (text-indent)

La propiedad text-indent permite definir la sangría de la primera línea del texto. Se puede asignar con valores de longitud (cm, in, px, etc.) o porcentajes respecto al ancho del elemento.

Elementos flotantes

La propiedad float admite como valores right, left y none que permite situar el elemento a la derecha, izquierda o en el lugar que aparezca definido.  Un ejemplo:

          img {float: right;}

La propiedad clear complementa a float y permite bajar el elemento, generalmente texto, hasta que el margen izquierdo esté libre, o el derecho o ambos. Los posibles valores son:  right, left, both o none.

Espaciado de letras y palabras

Las propiedades letter-spacing y word-spacing permiten especificar el espacio entre letras y palabras respectivamente. En ambos casos se puede especificar un valor específico (2.em, 3px, etc) o uno relativo; un valor negativo estrecha el espaciado y uno positivo lo amplía.

La propiedad white-space define cómo se gestionan las secuencias de espacios en blanco dentro de un elemento y los posibles valores son:

  • normal
  • nowrap
  • pre
  • pre-line
  • pre-wrap
  • inherit

Mayúsculas y minúsculas (text-transform)

La propiedad text-transform permite modificar el texto. Los valores que puede tomar son:

  • capitalize.- fuerza que la primera letra de cada palabra sea mayúscula.
  • uppercase.- fuerza todo el texto a mayúsculas.
  • lowercase.- fuerza todo el texto a minúsculas.

Decoración de texto (subrayar, tachar, etc)

La propiedad text-decoration puede tomar los siguientes valores:

  • none.- valor por defecto.
  • underline.- subrayado.
  • overline.- línea por encima.
  • blink.- parpadeo.
  • line-through.- tachado.

La propiedad text-shadow permite crear un efecto de sombra. El formato de esta propiedad es el siguiente:

          text-shadow: [color] distancia_horizontal distancia_vertical [, ...]

Como se puede observar puede repetirse, separando con una coma, los valores relacionados con cada capa de la sombra (color, dist. horizontal y dist. vertical). El color es opcional y las distancias son valores en unidades de longitud (px, pc, em, px, etc.).

Propiedades CSS vs. atributos en tablas HTML

Aunque la tendencia es el uso de CSS, aún existe mucho código de presentación en tablas. A continuación se muestra la equivalencia entre ambos enfoques:

ASPECTO ATRIBUTO HTML PROPIEDAD CSS
Alineación align text-align
valign vertical-align
Ancho width width y table-layout
Bordes border border
Espaciado en celdas cellpadding padding
Espaciado entre celdas cellspacing border-spacing

Para definir los bordes de una tabla y sus elementos dependientes se pueden utilizarlas propiedades border. Por ejemplo:

table {border: 3px solid black;}

La propiedad border-spacing permite especificar la distancia entre dos celdas consecutivas, especificando el espaciado horizontal y el vertical. Por ejemplo:

          table  {border-spacing: 3px 3px;}

La propiedad padding permite ajustar el espaciado entre el contenido y el borde.

La propiedad border-collapse permite eliminar el espacio entre los bordes asignando el valor collapse. El valor por defecto es separate. Por ejemplo:

          table, table *  {border-collapse: collapse;}

La propiedad empty-cells permite controlar el comportamiento de las celdas vacías. Admite los valores:

  • show.- muestra la celda aunque esté vacía.
  • hide.- oculta la celda si está vacía.

La propiedad table-layout determina cómo hace el agente de usuario para dimensionar la tabla. Puede tener los valores:

  • auto.- valor predeterminado. El ancho de la columna se ajusta al necesario para mostrar el elemento más ancho (teniendo en cuenta si se puede separar en varias líneas).
  • fixed.- fijo. El diseño horizontal solo depende del ancho de la tabla y del ancho de las columnas, no del contenido de las celdas.
  • inherit.- aceptado solo en algunos navegadores. El valor de la propiedad se hereda del elemento padre.

Las tablas pueden tener títulos definidos por la etiqueta <caption> y CSS tiene la capacidad para establecer la alineación y el posicionamiento de la leyenda del título mediante la propiedad caption-side, que tiene como valores posibles: top (arriba), bottom (abajo) o inherit (heredado). La alineación de la etiqueta caption se controla mediante la propiedad text-align o vertical-align.
