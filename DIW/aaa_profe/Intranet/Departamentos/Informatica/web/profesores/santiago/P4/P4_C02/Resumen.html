Instalación en MaX (Ubuntu)
    Es habitual trabajar con Linux y la trilogía Apache, MySQL y PHP, lo que ha dado en llamarse LAMP. En Windows también se puede instalar WAMP aunque puede optarse por usar IIS en vez de Apache.

Procedimientos de instalación

    Pueden instalarse los paquetes de forma individual o bien a través de un paquete que integre LAMP.

    PHP

    PHP se puede descargar de www.php.net/downloads.php o desde http://windows.php.net/snapshots

    MySQL

    MySQL es el servidor de base de datos preferido para el entorno PHP - Apache y que permite que las aplicaciones web puedan acceder rápida y flexiblemente a los datos almacenados. Puede descargarse desde www.mysql.com, siendo la página actual de descarga http://dev.mysql.com/downloads/ o http://downloads.mysql.com/snapshots.php.

Paquetes integrados

    Los paquetes integrados integran toda la instalación de Apache, MySQL y PHP en una sola herramienta. Esta característica facilita la tarea de instalación ya que todo se realiza en un único paso y las aplicaciones quedan configuradas para su funcionamiento inmediato.

    Hay varios paquetes integrados disponibles, todos gratuitos y con una configuración estandar que sirve para el uso habitual:
  • AppServ 2.6
  • AppServ 2.5.10
  • XAMPP 1.7.4
  • Wampserver 2.1
    Cada uno de estos paquetes está compuesto por diferentes versiones y utilidades que pueden consultarse en la pág. 605.

Instalación en MaX (Ubuntu)

    En primer lugar puede comprobarse si está instalado y ejecutándose Apache. Una de las formas más sencillas de comprobación es abrir el navegador y especificar la dirección: "localhost". Deberá aparecer una página web, que si no se ha modificado corresponde al fichero /var/www/index.html.

    Para comprobar si está instalado PHP puede modificarse dicha página introduciendo el código del capítulo 1 y cambiando la extensión de la misma por .php. A continuación habrá que especificar en el navegador la dirección http://localhost/index.php. La página deberá contener el texto correspondiente introducido desde PHP. Si no es así, seguramente habrá que instalar PHP .

    En www.php.net se encuentra la última versión estable (PHP 5.3.10), del 2 de febrero de 2012. Desde Synaptic la versión accesible es la 5.3.2.

    Error común: el navegador muestra una página en blanco

    Este error, bastante común, está descrito en la pág. 606.
