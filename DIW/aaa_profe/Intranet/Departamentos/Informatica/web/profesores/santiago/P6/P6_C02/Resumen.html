    Aunque el entorno tecnológico y social se encuentra sometido a un cambio constante, existen ciertos conceptos relativos a SEO que permanecen invariables y que podemos considerar fundamentales.

Arañas y robots

    Las arañas y robots son los elementos encargados de recoger la información que alimenta las bases de datos de los buscadores. Son procesos que recorren la Web para obtener información relevante en base a distintos algoritmos. Cada buscador tiene su propio proceso robot que pasará por los diferentes sitios web con una frecuencia determinada y aplicando ciertos criterios de prioridad que permitirán analizar los diferentes sitios con diferentes frecuencias.
    Es importante que el sitio web esté disponible el máximo tiempo posible para que los robots puedan actuar sobre él.
    El motor de búsqueda es un software especializado que se encarga de recuperar de sus bases de datos los fragmentos de texto que los robots y otros métodos han proporcionado.
    El proceso de búsqueda se inicia a partir del texto introducido en el cuadro de texto del buscador, aplicando algoritmos secretos en constante evolución.

Algoritmos secretos

    Los buscadores no revelan sus criterios de búsqueda ni en qué momento los cambian, y son distintos para los distintos motores. Los algoritmos utilizan diversos criterios: palabras clave, presencia de palabras clave en el título de la página, repetición de palabras clave en el texto del contenido, cantidad de enlaces entrantes de calidad, etc. Cada uno de los elementos de decisión tiene asignado un peso o importancia que valorado según una fórmula dará lugar a un posicionamiento.
    Si los criterios y algoritmos de búsqueda fueran conocidos, muchos SEO manipularían los sitios web para asegurarse un buen posicionamiento.
    Hay que tener encuenta los modos de funcionamiento de Google, Yahoo, etc. pero pensar en cómo llegar al cliente que nos quiere encontrar es algo que siempre da buenos resultados.

Contenido de texto

    El texto HTML incluido en la página web tiene una importancia fundamental para los robots y posteriormente para los motores de búsqueda. Los robots trabajan con texto; una imagen o un texto dibujado dentro de una imagen no tiene utilidad para ellos.
    De las imágenes es útil el nombre del archivo (si es descriptivo) y el texto alternativo asociado a las mismas.
    Es importante que en el texto se haga referencia a las palabras clave codificadas con la etiqueta "meta keywords" pues es valorado de forma positiva por los buscadores.
