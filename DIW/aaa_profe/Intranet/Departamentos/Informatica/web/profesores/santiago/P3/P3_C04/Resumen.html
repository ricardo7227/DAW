Necesidad del uso de arrays

    Uno de los mecanismos más utilizados en programación para gestionar grupos de objetos es el uso de vectores, matrices, tablas o arrays, que son elementos de datos accesibles mediante un índice.

    El bucle for es ideal para navegar por una tabla de elementos variando automáticamente el valor del índice.

    JavaScript implementa un objeto intrínseco, el objeto Array(), y solo es necesario crear una instancia para utilizarlo.

Objeto Array()

    El objeto Array() es un objeto predefinido que permite crear una tabla utilizando la sintaxis:

          var meses = new Array();
          meses[0] = "Enero";
          meses[1] = "Febrero";
          . . .
          meses[11] = "Diciembre";

    Otra posibilidad es:

          var meses = new Array ("Enero", "Febrero", ... "Diciembre");

    La propiedad length del objeto permite conocer el número de elementos de la tabla.

    Hay que tener en cuenta que las tablas en JavaScript son objetos (no variables) y una instrucción del tipo:

          var miTabla2 = miTabla1;

    duplica el nombre (referencias) de la tabla de datos pero al modificar cualquiera de ellos se modifica también el otro.

    Tablas asociativas

    Pueden definirse tablas con direccionamiento asociativo, es decir, aquellas en las que los índices no son números sino claves de texto. Un ejemplo:

          var meses = new Array();
          meses["Enero"] = "Los Reyes Magos";
          meses["Febrero"] = "El mes mas corto";
          . . .
          meses["Diciembre"] = "Vacaciones de Navidad";

    Este tipo de matrices no se puede recorrer de modo secuencial mediante el incremento o decremento de un índice; hay que utilizar la sentencia "for ... in" vista en el tema anterior para recorrer las propiedades de un objeto. Continuando con el ejemplo anterior tendremos:

          for (var elem in meses) {
              document.write ("Indice = ", elem, "  ----  ", "meses[", elem, "] = ", meses[elem], "<br>");
          }

    cuyo resultado puede verse pinchando aquí.

    Tablas multidimensionales

    Un elemento de una tabla puede ser a su vez otra tabla, lo que da lugar a las tablas multidimensionales. Un ejemplo sería:

          var tabla = new Array();
          tabla[0]  = new Array (1, 8, 3);
          tabla[1]  = new Array (3, 0, 675);

    que puede identificarse como una tabla de dos dimensiones, con dos filas y tres columnas. Para recorrerla pueden utilizarse dos bucles for anidados de la forma:

          for (var fila = 0; fila < 2; fila++) {
              for (var col = 0; col < 3; col++) {
                  document.write ("tabla (", fila, ", ", col, ") = ", tabla[fila][col], "<br>");
              }
          }

Métodos del objeto Array()

    El objeto Array() tiene una única propiedad (length) pero tiene varios métodos que permiten su manipulación.

    Conversión a cadena (join())

    El método join() convierte una matriz en una cadena de caracteres formada por el contenido de cada uno de los elementos de la matriz, separados por uno o varios caracteres que se definen como parámetro del método. Un ejemplo:

          var meses = new Array ("Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", "Septiembre",                                  "Octubre", "Noviembre", "Diciembre");
          document.write (meses.join (" - "));

    El resultado puede verse pinchando aquí.

    Concatenación de matrices (concat())

    El método concat() devuelve una matriz resultante de la concatenación de otras dos. Un ejemplo:

          var meses1 = new Array ("Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio");
          var meses2 = new Array ("Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre");
          var meses  = meses1.concat(meses2);
          document.write (meses.join (", "));

    El resultado puede verse pinchando aquí.

    Eliminación del último elemento (pop())

    El método pop() elimina el último elemento de una tabla. Un ejemplo:

          var meses = new Array ("Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio"
                                 "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre");
          meses.pop();
          document.write (meses.join (", "));

    El resultado puede verse pinchando aquí.

    Inclusión de elementos por el final (push())

    El método push() permite añadir elementos al final de la tabla. Un ejemplo:

          var meses = new Array ("Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio");
          meses.push ("Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre");
          document.write (meses.join (", "));

    El resultado puede verse pinchando aquí.

    Inversión de la tabla (reverse())

    El método reverse() invierte los elementos de la tabla. Un ejemplo:

          var colores = new Array ("Blanco", "Rojo", "Verde", "Negro");
          colores.reverse();
          document.write (colores.join (", "));

    El resultado puede verse pinchando aquí.

    Eliminar el primer elemento (shift())

    El método shift() elimina el primer elemento de la tabla. Un ejemplo:

          var colores = new Array ("Blanco", "Rojo", "Verde", "Negro");
          colores.shift();
          document.write (colores.join (", "));

    El resultado puede verse pinchando aquí.

    Recortar una tabla (slice())

    El método slice() permite recortar o extraer un subconjunto de elementos de una tabla. Devuelve una tabla formada por los elementos comprendidos entre los dos índices pasados como parámetros, sin incluir el último. Un ejemplo:

          var meses = new Array ("Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio"
                                 "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre");
          var verano = meses.slice (6, 9);
          document.write (verano.join (", "));

    El programa, como puede verse pinchando aquí, muestra "Julio, Agosto, Septiembre", es decir, muestra meses[6], meses[7] y meses[8].

    Ordenar una tabla (sort())

    El método sort() ordena una tabla ascendentemente. Combinando sort() y reverse() se puede obtener la tabla ordenada descendentemente. Un ejemplo:

          var colores = new Array ("Blanco", "Rojo", "Verde", "Negro");
          document.write ("Antes de ordenar.....: ");
          document.write (colores.join (", ") + "<br>");
          colores.sort();
          document.write ("Despues de ordenar: ");
          document.write (colores.join (", ") + "<br>");

    El resultado puede verse pinchando aquí.

    Reemplazar el contenido de elementos de una tabla (splice())

    El método splice() reemplaza desde un elemento en adelante, un número determinado de elementos, con los valores especificados. Por ejemplo, para reemplazar en la tabla siguiente desde el elemento colores[1] (Rojo), dos elementos, por los valores "Marron" y "Naranja" se pondrá:

          var colores = new Array ("Blanco", "Rojo", "Verde", "Azul", "Negro");
          colores.splice (1, 2, "Marron", "Naranja");
          document.write (colores.join (", "));

    El resultado, como puede verse pinchando aquí, es que se muestra "Blanco Marron Naranja Azul Negro".

    Insertar elementos al principio de una tabla (unshift())

    El método unshift() inserta elementos al principio de la tabla. Un ejemplo:

          var colores = new Array ("Blanco", "Rojo", "Verde", "Negro");
          colores.unshift ("Elemento_0", "Elemento_1");
          document.write (colores.join (", "));
