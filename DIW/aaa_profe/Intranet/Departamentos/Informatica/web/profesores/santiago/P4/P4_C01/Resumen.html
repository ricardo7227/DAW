Antecedentes de PHP

Novedades de PHP 6
    PHP significa Hypertext Preprocessor, es decir, preprocesador de hipertexto. PHP es un lenguaje de código abierto que se ejecuta en el servidor y que se incrusta en las páginas HTML para lograr que éstas tengan contenido dinámico basándose generalmente en información que se recupera desde bases de datos. Las páginas con código PHP se procesan en el servidor para producir la página que se envía al cliente. Un ejemplo sería:

          <!DOCTYPE html>
          <html>
            <head>
              <title>Cómo se incrusta el código PHP en una página HTML</title>
            </head>
            <body>
              <?php
                echo "Esto saldrá en el navegador";
              ?>
            </body>
          </html>

    PHP también actúa como elemento de conexión de las páginas web con los motores de bases de datos.

    PHP es un lenguaje de código abierto y de uso gratuito.

Dónde se utiliza PHP

    PHP se ejecuta sin cambios en una gran variedad de sistemas: servidores IIS de Windows, servidores Apache, en entornos Unix, Linux; Windows, Mac OS, etc.

PHP 5 o PHP 6

    En este texto se tratará la versión PHP 5.3 que es la versión estable y de uso extendido en los servidores comerciales. También se verán las novedades de PHP 6.

Para qué se utiliza PHP

    Las secuencias de comandos de PHP se utilizan de tres formas:
  • Desarrollo de código que se ejecuta en el servidor.- esta opción, la más extendida, genera código HTML y precisa de un servidor y de un navegador.
  • Desarrollo de aplicaciones GUI para ejecutar en el cliente.- PHP con el añadido del kit de desarrollo GTK se convierte en una herramienta adecuada para la programación de código que se ejecuta en el navegador.
  • Secuencias de comandos para utilizar directamente en el equipo sin necesidad de navegador ni servidor para resolver distintos tipos de tareas.

Código en el lado del servidor

    PHP (como ASP.NET) pertenece al grupo de lenguajes que se ejecutan en el lado del servidor. Es independiente del cliente y tiene capacidad para contactar con distintos tipos de servidores (Bases de datos, SMTP, Web, Directorios, ...), procesar datos y formularios, generar páginas de modo dinámico en base a resultados de consultas en bases de datos, etc.

Bases de datos que se utilizan con PHP

    Uno de los puntos fuertes de PHP es su conectividad a los principales sistemas de gestión de base de datos: MySQL, Ingres, Oracle, SQLite, Adabas, dBase, Empress, etc. etc.

Qué produce PHP

    PHP tiene una salida resultante clásica que es código HTML, es decir, el proceso de las páginas PHP en el servidor habitualmente genera como salida páginas HTML. Pero ésta no es la única posibilidad, también se pueden producir imágenes, archivos PDF, archivos Flash y cualquier clase de archivos de texto como XHTML o XML. Al procesar el servidor el código del ejemplo anterior producirá la siguiente salida, que es la que llega al navegador:

          <!DOCTYPE html>
          <html>
            <head>
              <title>Cómo se incrusta el código PHP en una página HTML</title>
            </head>
            <body>
              Esto saldrá en el navegador
            </body>
          </html>

    Como se puede comprobar, al navegador solamente llega código HTML, no llega nada de código PHP.

PHP no es sólo un lenguaje más

    Algunas de las ventajas que le han valido su popularidad son:
  • Fácil de aprender.
  • Incorpora gran cantidad de funciones.
  • Amplia disponibilidad de secuencias de comandos pregenerados en PHP.
  • Gran capacidad de incrustar código PHP en las páginas HTML.
  • Simplicidad de su vinculación con las principales bases de datos.
  • Compatible con las plataformas importantes.
  • Código fuente abierto y gratuito.
  • Soporta los principales protocolos (POP3, IMAP, LDAP).
  • Soporte para Java, COM, CORBA.
  • Soporte para XML con DOM y SimpleXML.

Antecedentes de PHP

    PHP nace en 1.994. Junto con Apache y MySQL se convierte en una combinación de software abierto muy utilizado. La versión 5.3.6 se distribuyó en marzo de 2.011 y actualmente está disponible la versión PHP 6.0.0.

Novedades de PHP 6

    Las novedades de PHP 6 con respecto a las anteriores se pueden resumir en:
  • Soporte para Unicode.
  • Mejoras en la seguridad.
  • Eliminación de la funcionalidad de magic_quotes.
  • Definición de namespaces.
  • Mejoras en las interfaces orientadas a objetos.
