Agrupación del texto


    La etiqueta <hr/> genera una línea horizontal.

    La etiqueta <pre> . . . </pre> permite colocar texto preformateado.

    La etiqueta <blockquote> hace un sangrado al bloque de texto para diferenciarlo del resto. Admite el atributo cite para indicar la fuente relacionada al texto.

    Listas

    Las etiquetas figure y figcaption

    Son nuevas en HTML 5 y permiten englobar una imagen (u otro tipo de contenido) y una información adicional o leyenda. Ver Fig. 1.



    Etiqueta div

    Esta etiqueta no tiene un significado concreto, se usa para crear divisiones en el documento a las que se pueden aplicar características independientes. Utiliza los atributos comunes, siendo los más usados: class, lang y title.
  • class.- asigna una clase al bloque.
  • lang.- define un idioma para el bloque.
  • title.- asigna un nombre al bloque que se hace visible al pasar el ratón sobre el bloque.
    HTML 5 no admite el atributo align para esta etiqueta.

    En HTML 5 esta etiqueta debe utilizarse como último recurso, cuando no hay otra etiqueta adecuada para resolver un caso puesto que ésta no aporta nada a la semántica del contenido.
