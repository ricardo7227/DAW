    ¿ Qué es CSS ?

    CSS viene de Cascade Style Sheets, es decir, hojas de estilo en cascada, y es una codificación que permite asignar estilo a los documentos HTML. W3C creó el estándar CSS y la versión actual recomendada es la 2.1, encontrándose la versión CSS 3 en desarrollo, con una implementación parcial en los navegadores.

    A partir de HTML 4 toda la codificación de estilo pudo separarse de la codificación HTML y almacenarse en archivos independientes. Gracias a CSS un mismo documento HTML se puede visualizar aplicando diferentes estilos, dependiendo, por ejemplo, del medio utilizado.

    Cuando se utiliza CSS, una etiqueta HTML cualquiera, por ejemplo, <h1> o <p>, no debería incluir información sobre cómo será visualizado su contenido, solamente debe servir para acotar el texto al que se aplica. La información de estilo deberá incluirse en una hoja de estilo.

    Interrelación HTML y estilos

    Ver ejemplo de pag. 203. Comprobar que el estilo está definido dentro de la etiqueta <head> y que cada vez que se quiere hacer referencia al estilo definido se utiliza el atributo class, de la forma:

    <p class="cabecera"> . . . </p>

    Hay tres formas de definir los estilos:
  • Definición en linea.- dentro de cada etiqueta.
  • Hoja de estilos interna.- dentro de la etiqueta head.
  • Hoja de estilos externa.- accesible con link.
    La hoja de estilo interna tiene el siguiente aspecto:

          . . .
          <head>
             <title>Primer ejemplo CSS</title>
             <style type="text/css">
                p.cabecera {font-weight: bold; font-style: italic; font-size: small;}
             </style >
          </head>
          . . .

    El formato de la etiqueta style permite indicar el tipo MIME y el medio de destino de la forma:

          <style type=" Tipo MIME " media=" Medio destino ">

    El atributo type por lo general es "text/css" pero el media admite los valores siguientes:
  • all.- todo tipo.
  • aural.- combinación de síntesis de voz y efectos de sonido.
  • braille.- dispositivos de este tipo.
  • embossed.- dispositivos de salida braille.
  • handheld.- dispositivos de este tipo, pantallas pequeñas y ancho de banda limitado.
  • print.- para impresora o previsualizaciones.
  • projection.- dispositivos de proyección.
  • screen.- pantalla.
  • tty.- dispositivos de caracteres fijos o teletipos.
  • tv.- televisión.
    Un documento puede incluir varios estilos orientados para los diferentes medios.

    La hoja de estilo externa es un documento externo con las definiciones de estilo, que se enlaza al documento HTML mediante una etiqueta link:

          <head>
            <link rel="stylesheet" type="text/css" href="miestilo.css"/>
             . . .
          </head>

    La gran ventaja de éste método es que una sola hoja de estilos puede ser utilizada por (y afectar a) múltiples documentos HTML.

    La definición en línea es la utilizada en la parte 1 del libro, con las etiquetas incluyendo el estilo.

    ¿ Por qué se denomina hoja de estilo en cascada ?

    La denominación "en cascada" proviene del hecho de que las definiciones pueden ser múltiples y una definición puede reemplazar a otra previa. El estándar CSS define las siguientes reglas para determinar qué estilo debe utilizar cuando se encuentra con casos de múltiples definiciones:
  • Si no hay definiciones se utiliza la hoja de estilo estándar del navegador.
  • Las definiciones de hoja de estilo externa (<link ...>) reemplazarán a las definiciones predeterminadas del navegador.
  • Las definiciones de hoja de estilo interna (<style>) reemplazarán a las anteriores.
  • Las definiciones de estilo en línea reemplazarán a las anteriores.
  • Si en algún nivel se utiliza la notificación "!important", esa declaración prevalecerá sobre el resto.
    Un ejemplo de uso de !important puede ser:

          . . .
          th {background-color: #ADFFF0 !important;}
