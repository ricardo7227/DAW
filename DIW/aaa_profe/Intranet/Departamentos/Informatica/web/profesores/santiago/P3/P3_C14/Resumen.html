DHTML (HTML dinámico) es la base de la manipulación dinámica de los componentes de una página para conseguir cambios del aspecto estético, de las posiciones y del contenido. Como de costumbre, no existe compatibilidad completa con todos los navegadores.

DOM: modelo de objeto document

    Los componentes de una página web se pueden visualizar como una estructura jerárquica y eso es básicamente el modelo de objeto documento (DOM). Cada uno de los elementos se trata como un nodo dentro de la estructura del documento.

    Existen varias herramientas que permiten el análisis y manipulación de un documento DOM, por ejemplo Dom Inspector de Mozilla (Herramientas - Web Developer - Dom Inspector).

    Dom Inspector permite modificar la página abierta por el navegador, no la del servidor, y la forma de trabajar es la siguiente:
  • 1. Abrir con Firefox una página y seleccionar Herramientas - DOM Inspector. También se puede especificar la URL en la barra de dirección del Inspector o a través de la opción File - Inspect a URL.
  • 2. Se puede visualizar la página pulsando sobre el botón "Inspect". Puede analizarse la página haciendo uso de los diferentes botones y opciones.
  • 3. Una vez encontrado el nodo a analizar, puede inspeccionarse en una ventana nueva.
  • 4. Puede guardarse la versión modificada como un documento nuevo.
    Las posibilidades de DHTML son muchas, por ejemplo:
  • Controlar la visibilidad de un elemento.
  • Mover o deslizar un elemento.
  • Cambiar los parámetros de estilo de un elemento.
  • Modificar el contenido de un elemento.

Objeto document

    El objeto document es el elemento fundamental de la programación DHTML dado que es el que permite acceder y modificar todos los elementos de una página. Contiene varios objetos (subobjetos) y algunos métodos orientados a la manipulación de los elementos integrados en document.

    Para verificar que un navegador permite acceder al modelo DOM (Document Object Model) puede utilizarse el método getElementById() de la forma:

          if (document.getElementById()) {// OK}
           else {alert ("Su navegador es demasiado antiguo");}

    Principales métodos del objeto document
  • getElementById(id).- permite obtener un elemento utilizando su identificador (atributo id).
  • getElementsByName(nombre).- permite obtener una tabla de elementos mediante el atributo nombre.
  • getElementsByTagName(etiqueta).- permite obtener una tabla de elementos usando como atributo una etiqueta.
    Contenido HTML de un elemento

    DHTML, haciendo uso de código JavaScript, permite modificar el contenido HTML de un elemento. La propiedad innerHTML contiene el código HTML del elemento y es una propiedad accesible en lectura y escritura. Por ejemplo, para modificar el contenido del elemento con atributo id "Texto" puede utilizarse:

          document.getElementById("Texto").innerHTML = "Otro texto";

Clase style

    La clase style permite modificar dinámicamente la presentación de los elementos de un documento. La hoja de estilo en cascada (CSS) permite definir el formato de un elemento y su codificación actúa por combinación acumulativa de las definiciones previas.

    Hemos visto que el estilo puede incorporarse dentro de las etiquetas HTML, o bien mediante la etiqueta <style> o utilizando ficheros CSS.

    Propiedad className

    La propiedad className del elemento del documento tiene el mismo efecto que el atributo class de la etiqueta HTML correspondiente y puede ser accedida en lectura y en escritura. Para poder utilizarla es obligatorio que el elemento HTML tenga definido el atributo class.

  • Aspecto resaltado.- si el ratón se encuentra sobre la fila.
  • Aspecto seleccionado.- si el elemento ha sido seleccionado.
    Propiedades de estilo

    Las propiedades de estilo permiten acceder y modificar el estilo de un elemento (o conjunto de elementos) desde código de JavaScript.

    Visibilidad

    Para controlar la visibilidad de un elemento en el documento existen dos propiedades:
  • visibility.- puede tomar los valores: visible o hidden.
  • display.- puede tomar los valores: none (oculto), block, inline o vacío.
    La ventaja de "display: none" (frente a "visibility: hidden") es que el elemento no ocupa lugar.

    Para acceder a las propiedades de estilo CSS desde JavaScript se debe acceder (obligatoriamente) a la propiedad correspondiente de estilo del elemento desde el objeto style. Un ejemplo para acceder a la propiedad display es el siguiente:

          document.getElementById("miId").style.display = "none";

    Posicionamiento

    La propiedad de estilo position permite establecer si el posicionamiento es relativo a otro elemento o absoluto con respecto a la esquina superior izquierda de la ventana (valores relative y absolute). Un ejemplo que hace uso de estas propiedades puede verse pinchando aquí.

    Dimensionamiento

    Las dos propiedades que controlan el dimensionamiento de un elemento en la ventana son width (ancho) y height (alto). Estos dos valores son modificables desde JavaScript si han sido definidos previamente con alguna dimensión.

    Elementos superpuestos

    Los elementos de una página se pueden superponer mediante la propiedad CSS z-index, que mediante código JavaScript se referencia con zIndex y que no debe asignársele valores negativos. Pinchando aquí puede contemplarse un ejemplo que permite modificar la superposición de elementos.

    Colores

    El color del texto y del fondo se definen mediante las propiedades CSS color y background-color. Su correspondencia con JavaScript son color y backgroundColor.

    Texto

    Para controlar el formato del texto, CSS incluye las propiedades: font-family, font-size, font-weight, font-style y text-decoration, que tienen su correspondencia en JavaScript con fontFamily, fontSize, fontWeight, fontStyle y textDecoration.

    Por otra parte, la etiqueta <a> tiene un tratamiento automatizado, sin necesidad de programar, respecto a algunas de las acciones del ratón, para marcar un estilo diferente para indicar un enlace ya visitado o que el ratón está pasando por encima del enlace. Ver ejemplo en pág. 552.

    Estilos de cursor

    Mediante código JavaScript se puede gestionar el formato del cursor utilizando la propiedad de estilo cursor. Los valores posibles para esta propiedad son:
  • default.- flecha normal.
  • text.- selección de texto.
  • progress.- indica el avance con un reloj de arena.
  • hand.- muestra una mano.
  • move.- indica un desplazamiento.
  • crosshair.- una cruz.
  • help.- indicador de ayuda.
  • wait.- reloj de arena.
  • n-resize.- redimensiona el sentido vertical.
  • e-resize.- redimensiona el sentido horizontal.
  • ne-resize.- redimensiona el sentido diagonal (norte-este y sud-oeste).
  • nw-resize.- redimensiona el sentido diagonal (norte-oeste y sud-este).
